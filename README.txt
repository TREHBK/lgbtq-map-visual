
📦 LGBTQ+ Cross-Cultural Expression Visualization – Offline Project Bundle

✅ 包含内容：
- index.html：主页面（含地图、时间线、词云）
- data.geojson：地图数据（英文版，Richard 位于墨尔本）
- timeline_offline.json：时间线事件（离线本地加载）
- README.txt：使用说明

🧭 使用方法：
1. 确保电脑安装 Python 3：https://www.python.org/downloads/
2. 解压此 ZIP 文件
3. 打开终端（命令行），进入解压后的文件夹目录
4. 输入命令启动本地服务器：
   python -m http.server 8000
5. 打开浏览器，访问：
   http://localhost:8000/index.html

✨ 所有内容无需联网即可本地演示！
